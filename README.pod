=head1 NAME

SocialMAP - A Facebook Connect integrated Geo-Social networking software. 

=head1 SYNOPSIS

	./socialmap daemon --listen http://localhost:3000

=head1 DESCRIPTION

Under the hood, SocialMAP uses the Perl based Mojolicious::Lite framework, the JavaScript
based Dojotoolkit and the JavaScript based Google Maps and Facebook Connect API's.
This carefully selected group of technologies allows for a feature rich interface where
people can easily interact with one another in ways they never have before using a unique
view into every day internet searching.

=head2 CPAN Setup

	sudo cpanp install Mojolicious::Lite DBIx::Class

=head2 SQLite3 Setup

	sqlite3 socialmap.db < lib/Model/Schema.sql
